# Development Guide

This document contains commands and instructions for building, running, and maintaining this Next.js project.

## Development Server
To launch the hot-reloading development server locally:
```bash
pnpm run dev
```

## Production Build
To verify type safety, run ESLint, and compile the optimized production bundle:
```bash
pnpm run build
```

## Production Server
To launch the compiled production bundle locally:
```bash
pnpm run start
```

## Code Quality Check
To check for ESLint violations:
```bash
pnpm run lint
```
